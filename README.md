# Addon-Pokedex-Azuriom
Azuriom Pokédex Addon is an HTML/PHP and JavaScript extension that generates a custom Pokédex inside an Azuriom website. It connects to a Minecraft server to fetch Pokémon JSON data files, then parses and imports them to dynamically populate the Pokédex with up-to-date entries.
